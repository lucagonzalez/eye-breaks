Install EyeBreaks (Not Notarized)

EyeBreaks is currently distributed without Apple notarization. macOS may show security warnings on first launch.

Download the DMG, open it, and drag EyeBreaks.app to Applications.

Download Latest DMG
  1. Double-click EyeBreaks-latest.dmg and drag EyeBreaks.app to Applications.
  2. Right-click /Applications/EyeBreaks.app and choose Open.
  3. If macOS still blocks launch, open Terminal and run:
